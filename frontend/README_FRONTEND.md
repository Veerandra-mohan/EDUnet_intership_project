# DeepInspection AI - Frontend

The frontend is a custom React Single Page Application (SPA) built with Vite. It features a premium, responsive Dashboard equipped with Light/Dark mode transitions without relying on massive UI libraries.

## Key Technologies
- **React 19**
- **Vite**
- **React Router DOM** (Navigation Architecture)
- **Lucide React** (Clean SVG Iconography)
- **Vanilla CSS** (Custom CSS Variables for dynamic theme switching)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Navigate to `http://localhost:5173`.

## UI Structure
The system is built on a Sidebar + Outlet architecture.
The UI implements advanced glassmorphism and subtle glowing aesthetics for a modern, high-tier SaaS feel.
